# CareCredit Expense Tracker

## Description
The CareCredit Expense Tracker is a specialized financial management tool designed specifically for users navigating healthcare costs and medical financing. In the complex world of medical billing, keeping track of out-of-pocket expenses, dental procedures, veterinary bills, and cosmetic treatments can be overwhelming. This application provides a streamlined interface to log medical transactions, monitor remaining balances, and categorize expenses according to healthcare needs. By centralizing your CareCredit usage, you can gain a clearer picture of your health-related debt and payment timelines, ensuring you take full advantage of promotional financing periods without the stress of disorganized paperwork. Whether you are managing a long-term treatment plan or a one-time emergency procedure, this tool acts as your digital financial assistant, bringing transparency and organization to your healthcare journey.

## Features
* **Transaction Logging:** Easily input medical service dates and costs.
* **Category Management:** Organize expenses by provider type (Dental, Vision, Vet, etc.).
* **Real-time Totals:** Dynamically view your total healthcare spend.
* **Responsive Design:** Access your data on mobile or desktop.

## Technologies Used
* **React.js** - Frontend Framework
* **CSS3** - Custom Styling
* **JSON** - Data Storage Simulation

## Getting Started
### Clone the Repository:
`git clone https://github.com/jerrymarlowe451/carecredit-tracker.git`

### Navigate to the Project Directory:
`cd carecredit-tracker`

### Install Dependencies:
`npm install`

### Start the Development Server:
`npm start`
